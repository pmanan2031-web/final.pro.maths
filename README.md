# Loan Default Risk Analysis using Python

## Project Overview

This project performs statistical analysis and linear algebra operations on a simulated loan dataset using Python libraries such as NumPy, Pandas, SciPy, and Matplotlib.

The project demonstrates:

- Dataset generation
- Statistical analysis
- Probability calculations
- Data visualization
- Linear algebra applications

---

# Technologies Used

- Python
- NumPy
- Pandas
- SciPy
- Matplotlib

---

# Dataset Information

The dataset contains 5000 customer loan records with the following attributes:

| Column Name | Description |
|-------------|-------------|
| customer_id | Unique customer ID |
| age | Customer age |
| income | Customer annual income |
| loan_amount | Loan amount |
| credit_score | Credit score |
| loan_term | Loan duration in months |
| default_status | Loan default status (1 = Yes, 0 = No) |

---

# Features Implemented

## 1. Central Tendency & Dispersion

The following statistical measures are calculated:

- Mean
- Median
- Mode
- Range
- Variance
- Standard Deviation

---

## 2. Probability & Events

The project calculates:

- Probability of loan default
- Conditional probability
- Contingency table

---

## 3. Distribution & Visualization

Visualizations included:

- Histogram of credit scores
- Gaussian distribution curve
- Q-Q Plot

Statistical shape analysis:

- Skewness
- Kurtosis

---

## 4. Linear Algebra Applications

Vector operations performed:

- Dot Product
- Vector Norm
- Angle Between Vectors
